# Forgotten Domain — Multi-Planet Strategy Game

A browser-based strategy game with 5 planets, 47 unique creatures,
real-time sigil crafting, creature summoning, enemy AI, and combat.

## Getting Started

```bash
npm install
npm start
```

Then visit `http://localhost:3000/game.html`

Or use the menu at `http://localhost:3000/menu.html`

## Tech

- Express server with JWT auth and SQLite
- SVG-based map with interactive bases
- Creature summoning and sigil crafting systems
- Enemy AI with autonomous decision-making
- Combat system with creature effects and auras
- 5 planets: Voxya, Orilyth, Korvess, Sanguis, Silith-9
