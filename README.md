# Forgotten Domain — Voxya: The Shattered Archipelago

A browser-based strategy game built with vanilla HTML, CSS, and JavaScript.

## Getting Started

Open `game.html` in a browser or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080/game.html`

## Tech

- Pure frontend — no frameworks, no build step
- SVG-based map with interactive provinces/bases
- Creature summoning and sigil crafting systems
